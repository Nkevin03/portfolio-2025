# Creazione di una Todo App con Next.js  

Questa è una **todo app** sviluppata con **Next.js**, **TailwindCSS**, **TypeScript**, **Prisma** e **Supabase**.  


## Descrizione  

L'obiettivo di questo progetto era imparare a gestire i dati con **Supabase**, sfruttando **Next.js 15** e **TailwindCSS**. Ho progettato un'interfaccia semplice e minimalista, dando priorità alla leggibilità e alla user experience.  

Ho iniziato sviluppando una CRUD per la gestione delle attività, includendo diversi campi per filtrare e organizzare le task in base a vari criteri.  


![Gestionale](/assets/work/gestionale/add_task.jpeg)


Successivamente, ho implementato una sezione dedicata alla gestione dei clienti e dei loro dati.  


![Gestionale](/assets/work/gestionale/add_client.jpeg)  

## Caratteristiche Principali  

- **Supabase** per la gestione del database  
- **TypeScript** per una tipizzazione più sicura  
- **Shadcn/ui** per componenti moderni e stilizzati  


## Stack Tecnologico

* Next.js 14
* React
* TailwindCSS
* TypeScript
* Framer Motion
